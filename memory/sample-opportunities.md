## Active Opportunities

### AI Workflow Guide for Non-Technical Buyers
**Flagged:** 2026-03-12
**Source:** reports/2026-03-12-research.md
**Status:** Pursuing

**What it is:**
A digital product teaching solo operators how to use multiple
AI models as a coordinated team. No coding required. GitHub
as shared memory.

**Why it's interesting:**
Competing solutions require Supabase, MCP servers, and Vercel —
significant technical overhead. A zero-code alternative has a
clear gap to fill. Gumroad comps suggest $24-$34 price point
is viable for this audience.

**Open questions:**
- Is there enough search volume to validate demand?
- What's the right hook for the Gumroad listing?

### Local AI Model Integration
**Flagged:** 2026-03-13
**Source:** analysis/2026-03-13-analysis.md
**Status:** Deferred

**What it is:**
Extending the multi-AI team framework to include locally-run
models like Ollama for users who want to keep sensitive data
off cloud services.

**Why it's interesting:**
Privacy-conscious buyers — legal, medical, finance — would
pay a premium for a version of this workflow that doesn't
send data to external APIs.

**Open questions:**
- How large is this audience relative to the general buyer?
- Does it warrant a separate product or an add-on?
